Slides Upgrader 是一个用于升级 STEM 课程材料的 skill。它面向质量较低、内容过时，有防自学设计的 PPTX/PDF 课件，把原始课程材料重构为更清晰、更适合本科生的学习资料，并按需导出 PPTX、PDF 或两者。

本项目是一套可被 AI agent调用的工作流。实际的 PPTX/PDF 解析、渲染、编辑和导出能力以及网络检索能力由运行环境提供。

## 主要功能

- 重建课件结构：把松散、跳跃或依赖口头讲解的课程内容整理为系统的内容结构。
- 补全学习缺口：补充必要的先修知识、符号解释、推导步骤、例题提示、练习答案或版本说明。
- 正确性检验：基于大模型和网络搜索结果，核查课件内容的可溯源性、技术正确性和时效性，修正错误内容。
- 提升阅读体验：改善文字层级、公式呈现、代码排版、图表说明和最终 PPTX/PDF 的整体可读性。

## 工作原理

Slides Upgrader 的主要工作流如下：

1. 确认输入格式、学科、目标语言、是否保留原视觉风格，以及用户需要的输出路线。
2. 解析源课件，保留原文、结构、图表、公式、代码和页码/幻灯片来源。
3. 将源内容整理成可分析的课程知识结构，明确概念、先修关系、例题、练习和不确定项。
4. 根据自学需求补全必要缺口，并标明补充内容的依据。
5. 对过时、版本相关或外部可验证的内容进行核查，必要时记录修正和来源。
6. 按用户指定的输出路线重建课件，保持源内容可追溯，同时改善学习顺序和视觉呈现。
7. 渲染并检查最终输出，确认没有空白页、文字重叠、公式损坏、代码不可读、占位符残留等问题。
8. 交付最终课件和质量报告。

## 安装方法

### 方式一：作为通用 skills 目录安装

将本项目目录放入你的 agent 运行时、自动化系统或 skills 管理器约定的 skills 目录中，例如：

```text
skills/
  slides-upgrader/
    SKILL.md
    references/
    assets/
    agents/
```

安装后，刷新或重启对应运行环境，使其重新发现 `slides-upgrader`。

### 方式二：在当前仓库中开发和测试

克隆仓库后，将仓库文件作为项目根目录

```text
slides-upgrader/
  SKILL.md
  README.md
  agents/
  assets/
  references/
  tests/
```
## 环境要求

本项目本身没有固定的包管理安装步骤，也没有内置运行时依赖清单。实际执行课件升级时，运行环境需要根据输入文件、输出路线和源内容提供相应能力。

### 基础能力：

- PPTX/演示文稿能力：用于 PPTX 输入或 PPTX 输出，至少应支持检查、重建、预览渲染、导出 PPTX/PDF。
- PDF 能力：用于 PDF 输入、PDF 输出和最终视觉验证，至少应支持文本提取、页面渲染、PDF 创建或导出。
- 检索或浏览能力：仅在需要核查时间敏感、版本相关或外部可验证事实时使用。
> 对于 PPTX 和 PDF 能力，建议安装面向 PDF 和 presentation 的 skills。

### 按内容触发的渲染能力：

- 复杂数学公式：MathJax、KaTeX 或 LaTeX；可选 `sharp`、`dvisvgm` 等用于转 SVG/PNG。
- 结构图：Graphviz、Mermaid 或等价图渲染器；适用于树、语法树、状态机、自动机、流程图、依赖图、DAG、知识图谱局部结构等。
- 数学示意图：Node.js 或 Python 生成 SVG；可选 `sharp`、CairoSVG、Inkscape 或浏览器用于转 PNG。
- 数据图和算法可视化：Python 的 Pillow、matplotlib、networkx，或 Node.js 的 Canvas、SVG、D3 等。

> 这些渲染能力是条件要求，不需要每次全部安装。运行工作流时，skill 会根据源材料和输出路线做环境预检；如果缺少必要能力，agent 应停止执行并报告缺失项。

## 使用方式

在支持 skills 的 AI agent 或自动化环境中，把待升级的 PPTX 或 PDF 提供给执行者，并说明期望输出格式即可。例如：

```text
使用 slides-upgrader 升级这份计算机科学课程 PDF，使其更适合本科生自学，并导出 PPTX 和 PDF。
```

也可以更明确地指定要求：

```text
请升级 lectures/week03_regex.pdf：
- 目标语言：中文
- 输出：PDF
- 保留原始术语和代码
- 补充缺失的先修知识、例题推导和练习答案
```
